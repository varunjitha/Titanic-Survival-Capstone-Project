# Titanic Survival Prediction and Passenger Analysis

## Problem Statement
Analyze passenger data from the Titanic disaster and build a machine learning model to predict survival outcomes.

## Objectives
- Clean and preprocess data
- Perform Exploratory Data Analysis
- Identify important survival factors
- Build a predictive model
- Generate insights and recommendations

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Methodology
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Selection
4. Logistic Regression Model
5. Performance Evaluation

## Results
- Model Accuracy: 80%
- Passenger class influenced survival
- Female passengers showed higher survival rates

## Recommendations
- Prioritize vulnerable groups during emergency evacuations
- Improve passenger safety protocols
- Use predictive analytics for risk assessment

## Conclusion
The project successfully analyzed passenger survival patterns and developed a machine learning model capable of predicting survival outcomes with approximately 80% accuracy.
